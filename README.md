A growing collection of small C++ practice projects built while following [Dr. Mohammed Abu-Hadhoud's Programming Roadmap](https://www.youtube.com/@dr.mohammedabuhadhoud). Each project focuses on a specific concept — from procedural programming basics to file handling, structs, and menu-driven console applications — as a foundation before moving into Object-Oriented Programming and beyond.

---

## 📖 About This Repository

This repo documents my learning journey step by step. Instead of jumping straight into big frameworks, I'm deliberately building small, focused console projects to solidify the fundamentals: logic, functions, data structures, file I/O, and clean program flow — the same foundations that later scale into full systems like [DVLD (Driving & Vehicle License Department)](#), my main desktop application project.

Every project here includes its own `README.md` with a full breakdown: features, code structure, sample input/output, concepts practiced, and known issues — written the way I'd want to hand off documentation in a real team.

## 🧠 Concepts Practiced Across the Repo

- Procedural programming (functions, control flow, pass-by-reference)
- Structs and enums for organizing related data
- File handling with `fstream` (reading, writing, appending)
- String parsing and custom serialization (splitting/joining records)
- Menu-driven program design with `switch` statements
- Input validation and basic access control logic
- Writing clean, professional documentation for every project
